# MorphoNews Evolution Features

## 進化の哲学 (Evolution Philosophy)

MorphoNewsは「進化（Evolution）」と「変化（Variation）」の2つの軸で成長します：

### 1. 機能の進化 (Feature Evolution) - 進化
毎回の更新で**新しい機能を追加**し、ユーザー体験を段階的に向上させます。

### 2. スタイルの変化 (Style Variation) - 変化
視覚的なデザインは変化しますが、過去のスタイルは保存され、ユーザーがいつでも選択できます。

---

## 現在の機能 (Current Features)

### 🎨 スタイル管理システム
- **6つのテーマ**: デフォルト、オーシャン、フォレスト、サンセット、ミッドナイト、チェリーブロッサム
- **スタイルギャラリー**: 全スタイルをプレビュー付きで閲覧・選択
- **設定の保存**: 選択したスタイルは自動保存され、次回訪問時も適用

### 📊 読書体験の向上
- **読書進捗バー**: ページ上部に進捗を表示
- **文字サイズ調整**: A+/A-ボタンで70%〜150%の範囲で調整可能
- **設定の永続化**: 文字サイズも自動保存

### 🏗️ アーキテクチャ改善
- **モジュラーCSS**: ベーススタイルとテーマを分離
- **外部CSS/JS**: メンテナンス性と拡張性を向上
- **メタデータ管理**: プロンプトやトークン情報をJSON形式で保存

## 技術スタック (Tech Stack)

- Google Fonts

### バックエンド
